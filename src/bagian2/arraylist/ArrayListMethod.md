| Method | Kegunaan |
| :--- | :--- |
| `add(elemen)` | Menambah elemen di posisi terakhir. |
| `add(i, elemen)` | Menyisipkan elemen pada indeks i. |
| `get(i)` | Mengambil elemen pada indeks i. |
| `set(i, elemen)` | Mengganti elemen pada indeks i. |
| `remove(i)` | Menghapus elemen pada indeks i. |
| `remove("X")` | Menghapus objek pertama yang bernilai "X". |
| `size()` | Mengembalikan jumlah elemen. |
| `contains(x)` | Mengecek apakah x ada (true atau false). |
| `isEmpty()` | Mengecek apakah list kosong (true atau false). |
| `clear()` | Mengosongkan seluruh list. |
